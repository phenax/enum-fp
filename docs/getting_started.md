# Enum-FP
Functional enum type for javascript with pattern matching

## Getting started
**To add this package to your project**
```bash
yarn add enum-fp
```
Or if you are one of those npm or pnpm nuts
```bash
npm i --save enum-fp
```
```bash
pnpm i --save enum-fp
```
**Import it to your file**
```javascript
import EnumType from 'enum-fp';
```
